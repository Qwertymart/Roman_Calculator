# Roman_Calculator


#### Roman_Calculator — это консольный калькулятор, поддерживающий арифметические операции с римскими числами. Он позволяет выполнять сложение, вычитание, умножение и деление, а также обрабатывать выражения с вложенными скобками.

* Поддержка римских чисел от I до MMMCMXCIX (1–3999).

* Арифметические операции: +, -, *, /.

* Поддержка скобок для определения приоритета операций.

* Обработка ошибок: деление на ноль, отрицательные и нулевые результаты, некорректные символы.

## Архитектура проекта
* `main.go`: Точка входа в приложение. Выполняет тесты и запускает интерактивный режим.
* `evaluation`: Обрабатывает токены и вычисляет результат выражения.
* `tokenizer`: Разбивает входную строку на токены (числа, операторы, скобки).
* `roman`: Преобразует римские числа в арабские и обратно, а также выполняет арифметические операции.


## Сложность алгоритма
Временная сложность: `O(n)`, где n — длина входного выражения. \
Пространственная сложность: `O(n)`, используется для хранения токенов и стека операций.

## Работа алгоритма
### Токенизация (разбор строки)
Цель: Преобразовать входную строку в список токенов: римских чисел, операторов и скобок.
Это реализуется с помощью посиввольного прохода по строке, исключая символы разделения.
### Обработка токенов и вычисление
Алгоритм вычисления выражения в инфиксной форме без конвертации в постфиксную запись.\
Есть 2 стека: для операции и операндов\
Посимвольно идём по строке\
Если символ - операнд, то кладём его в стек операндов\
Если символ “(”, то кладём в стек операций\
Если символ “)”, то выполняем все операции из стека, до встречи “(”, скобки удаляем\
Если символ - оператор, то выполняем, пока приоритет больше или равен, после текущий символ в стек операций\
Когда строка закончилась, из стека операции выполняем все операторы и результат выражения будет в стеке операндов.
